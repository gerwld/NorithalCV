NorithalCV Pure is a lightweight, minimal WordPress theme tailored for personal CVs and portfolios. Its simple yet modern design makes it easy to use, lightning-fast, and versatile, with built-in dark mode support.

== Description ==

**NorithalCV Pure** is designed for professionals and creatives who need a snappy, lightweight theme to showcase their skills and achievements. Whether you're creating a CV, portfolio, or minimalist blog, this theme is an excellent choice.  

Key features include:  
- **Ultra Lightweight**: Optimized for speed and performance.  
- **Dark Mode**: Built-in support for a dark theme to reduce eye strain.  
- **Snappy Performance**: Instant-loading pages for a smooth browsing experience.  
- **Minimal Codebase**: Easy to understand and customize for developers and beginners alike.  
- **Responsive Design**: Looks great on desktops, tablets, and mobile devices.  

== Installation ==

1. Upload the `norithal-cv` folder to the `/wp-content/themes/` directory.  
2. Activate the theme through the "Themes" menu in WordPress.  
3. (Optional) Customize the theme using the WordPress Customizer under "Appearance > Customize".  

== Features ==

- **Dark Mode Toggle**: Easily switch between light and dark themes.
- **Lightweight Framework**: Ensures minimal bloat for faster loading times.
- **Developer-Friendly Structure**: Modular files for seamless customization.
- **Compatible with Major Plugins**: Works flawlessly with popular plugins like Contact Form 7 and Yoast SEO.
